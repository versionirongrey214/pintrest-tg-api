## 📥 About the application
The pintrest-tg-api tool helps you save high-quality images from Pinterest directly to your computer or phone. Pinterest images often look small or pixelated when saved through a browser. This tool retrieves the original 4K or 8K file. 

You can use this tool with your Telegram account. The application acts as a bridge. It receives a link from you on Telegram and sends back the high-resolution image file. It handles the difficult parts of Pinterest interface protections behind the scenes. You do not need to manage code to use this service.

## 🛠️ Configuration
The software requires a small amount of setup to talk to your Telegram account.

1.  Find the file labeled setup.exe in your extracted folder.
2.  Double-click this file to launch the program.
3.  A settings window will appear on your screen.
4.  Paste your Telegram Bot Token into the marked box. If you do not have a token, you can request one from the BotFather inside Telegram.
5.  Paste your Pinterest session cookie into the second box. This cookie allows the software to view high-resolution files that normally require a login.
6.  Select the Save button.

## 📲 Usage
After you save the configuration, the application remains active in your system tray. Look for the small icon near your computer clock.

1.  Open the Telegram application on your phone or desktop.
2.  Search for your bot username.
3.  Send a message to the bot containing any Pinterest image link.
4.  Wait a few seconds while the bot processes the request.
5.  The bot will send the high-resolution image back to your chat window.
6.  Click the image and select Save to store it on your device.

## 💡 Troubleshooting
Most users experience success by following these steps. If you encounter issues, review these common solutions.

If the bot does not respond, check your internet connection first. Ensure that your computer stays connected to the network while the bot runs. Next, verify that your session cookie is correct. Pinterest periodically expires session cookies. If your bot stops working after a few weeks, fetch a new cookie from your Pinterest browser session and update it in the settings menu.

If the image looks low quality, verify the link. Ensure the link points directly to the Pinterest pin page and not a board or a search result. The tool works best with direct pin links.

If your antivirus software prevents the program from opening, this happens because the application communicates over the internet to perform its tasks. You can add an exception in your Windows Security settings to allow the program to function.

## 📁 Storage and Privacy
The application does not store your Pinterest credentials on a remote server. Everything happens on your machine. The session cookie remains local to your computer. Your Pinterest images will download to a specific folder within the application directory. You can change this path in the settings menu at any time.

The application deletes temporary files automatically after it sends the image to Telegram. This ensures your hard drive does not fill up with duplicate files. If you wish to save the images permanently, use the Save option within the Telegram application interface.
